1. UXPin - https://www.uxpin.com/

UXPin has a free trial that allows you to create interactive wireframes and user flows rapidly. It includes many built in elements for web/mobile/desktop products including bootstrap, it also allows you to standardize the design system. Allows you to create either static or interactive designs, and includes project collaboration with notes and tracking.

2. Balsamiq - https://balsamiq.com/products/mockups/UXPin/

Balsamiq lets you create mockups using either a “sketch skin” which imitates pencil sketches or a “wireframe skin” which looks more like an actual website. It supports web/mobile/desktop mockups and allows teams to collaborate on designs similar to UXPin. It also offers a free trial.

3. Axure - http://www.axure.com/

Axure features dynamic content, animations, math functions to dynamically change labels on the mockup. These mockups can then be shared easily through a link where the mockup is hosted on the cloud. It also supports collaboration, user flows and documentation. You can try a fully functional version of Axure for 30 days for free.

4. Fluid - https://www.fluidui.com/

Has 16+ in built libraries to pick from for the design patterns, supports gestures and transitions between screens as well as collaboration in teams. Completely done in-browser, easy to send previews to other team members. Has a free trial but seems to be focussed on mobile apps.

5. Pidoco - https://pidoco.com/en/

Drag and drop interface. Elements look basic like they don’t include common libraries such as bootstrap. Has a free trial, collaboration , sharing is done by exporting as PDF/RTF. Doesn’t seem to have dynamic content, i.e. not as powerful as the other options.

Conclusion

After looking at some of the available options for mock up tools, it seems UXPin, Balsamiq or Axure are our main options. They are all quite similar so it comes down to which interface we find easier to use and which option we have most experience with. As some of us have used Balsamiq in the past we may go with this option, however we will try out the others to see how they are as they all offer a free trial.
